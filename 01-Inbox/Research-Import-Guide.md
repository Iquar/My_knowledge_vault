# Research Import Guide

## 入口来源
- Web Clipper 捕获的网页
- MarkDownload 下载的 Markdown
- 手动创建的临时笔记
- 从外部工具导入的研究材料

## 导入规则
1. 所有原始材料首先进入 `01-Inbox/Downloaded`
2. 附件（图片、PDF）进入 `01-Inbox/Attachments`
3. 经过 Research Cleaner 处理后进入 `01-Inbox/Cleaned`
4. 最终分配至 `04-Research` 对应分类

## 文件命名规范
`YYYY-MM-DD_来源_主题关键词.md`

## 优先处理标志
- 🔴 紧急：24h 内处理
- 🟡 重要：3 天内处理
- 🟢 参考：1 周内处理
