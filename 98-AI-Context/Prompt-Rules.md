# Prompt Rules

## 通用规则
1. 使用中文进行所有笔记和研究
2. 保持 Markdown 格式规范
3. 每条笔记必须包含 frontmatter（tags, created, updated）
4. 一个文件一个主分类目录 + 多个主题标签

## Agent 使用规则
1. 需要具体输出时，明确指定格式
2. 研究类任务：保留原文，不总结，不改写观点
3. 创作类任务：指定平台、格式、受众

## 文件组织规则
- 遵循 folder + tag + link 三重组织
- 定期运行 Knowledge Base Audit
