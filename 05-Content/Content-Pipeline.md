# Content Pipeline Overview

## 内容生产流程
```mermaid
flowchart LR
    A[Ideas 收集] --> B[Content Brief]
    B --> C[Scripts 撰写]
    C --> D[编辑审阅]
    D --> E[Published 发布]
    E --> F[数据反馈]
    F --> A
```
